Configuración inicial — MOZI

Para guardar tus datos en Google Sheets necesitás crear un script backend. Son 3 pasos, tarda menos de 5 minutos.

Paso 1. Entrá a sheets.new y creá una planilla nueva llamada MOZI GrowyFunnels.
Paso 2. En la planilla: Extensiones → Apps Script. Borrá el código existente, pegá el contenido del archivo apps-script.gs. Guardá → Implementar → Nueva implementación → tipo Aplicación web → acceso Cualquier persona. Copiá la URL.
Paso 3. Pegá esa URL acá abajo y hacé clic en Guardar.
