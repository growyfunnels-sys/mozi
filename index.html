Conectá Google Sheets como backend. Son 3 pasos, menos de 5 minutos.

Paso 2. En la planilla → Extensiones → Apps Script → borrá todo → pegá el archivo apps-script.gs → Guardar → Implementar → Nueva implementación → Aplicación web → Cualquier persona → Implementar. Copiá la URL.
Paso 3. Pegá esa URL acá abajo.
