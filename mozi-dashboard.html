Configuración inicial — MOZI

Para guardar tus datos en Google Sheets necesitás crear un script que actúe como backend. Son 3 pasos, tarda menos de 5 minutos.

Paso 1. Entrá a sheets.new y creá una planilla nueva. Nombrala MOZI GrowyFunnels.
Paso 2. En la planilla, andá a Extensiones → Apps Script. Borrá el código que hay y pegá el código del archivo apps-script.gs que te generé. Guardá y hacé clic en Implementar → Nueva implementación → tipo Aplicación web → acceso Cualquier persona. Copiá la URL que te da.
Paso 3. Pegá esa URL acá abajo y hacé clic en Guardar.
